# cube3d — prosty projekt p5

To jest prosty szablon projektu w JavaScript używający biblioteki p5.js.

- Otwórz `index.html` w przeglądarce (lub użyj rozszerzenia Live Server).
- `sketch.js` rysuje metaliczną powierzchnię oraz wyśrodkowany, kwadratowy viewport z czarną przestrzenią.

Uruchomienie (PowerShell):

```powershell
start index.html
```

Lub w terminalu (Windows):

```powershell
npx http-server -c-1
```
